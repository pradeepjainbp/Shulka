# Contributing to Shulka

## Who can contribute

Shulka's application code is proprietary. External contributions to the app
are not accepted at this time.

**GST rule definitions** (`/rules/` directory) are CC0-licensed public domain.
Contributions there are welcome — see the contributor flow in Phase 8.

## GST rule contributions

Rules must cite a CBIC notification or circular. The format is defined in
`PHASES.md` (Phase 8 contributor flow). Rule PRs are reviewed by Pradeep Jain
before merge.

## Reporting bugs

Open an issue on GitHub: https://github.com/pradeepjainbp/Shulka/issues

Include:
- What you expected
- What happened instead
- Steps to reproduce
- Screenshots if UI-related

## Code of conduct

Be respectful. Discussions about GST rules should cite authoritative sources
(CBIC notifications, circulars, advance rulings). Speculation without citation
will be closed.
